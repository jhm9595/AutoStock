# AutoStock - 키움 REST API 기반 주식 자동 매매 대시보드 시스템

이 프로젝트는 **React(Vite) 프론트엔드**와 **FastAPI(Python) 백엔드**로 구성된 국내 주식 자동 매매 및 모니터링 시스템입니다. 키움증권의 새로운 REST API 및 WebSocket 명세를 바탕으로 자동 매매 봇 제어 및 실시간 상태 관리를 대시보드 형태로 제공합니다.

## 🔒 보안 강화 아키텍처
* **API Key 파일 격리**: 본 소스 코드나 환경 변수 파일(`.env`)에 기밀 키를 작성하여 커밋하지 않습니다. 로컬 PC 경로인 `config/` 디렉토리에 텍스트 파일(`*_appkey.txt`, `*_secretkey.txt`) 형태로 두고 백엔드가 로컬 파일 시스템에서 직접 읽어 인증을 시도합니다.
* **Git 추적 제외**: `.gitignore` 설정을 통해 실거래 계좌번호, 발급 토큰(`credentials.json`), 매매 설정 및 이력 데이터(`state.json`)는 GitHub 저장소에 절대 업로드되지 않습니다.

## 💻 대시보드 UI 구성 영역 (7대 영역)

프론트엔드 대시보드는 **흑과 백(Monochrome)을 주 색상**으로 활용한 현대적인 테크니컬 디자인을 적용하였으며, **수익률 지표는 빨강(이익), 파랑(손해)**으로 직관적으로 시각화했습니다.

1. **일반적인 정보 영역**: 계좌번호, 현재 요일, 로그인 타임스탬프, 평가 잔고 및 주문 가능 캐시 금액을 한눈에 표시하며, 실거래/시뮬레이션 모드 전환 기능을 제공합니다.
2. **조건검색 결과 목록**: 활성화된 조건검색 필터에 포착된 주식 목록을 실시간으로 노출합니다.
3. **조건 연동 토글 영역**: 키움증권에 등록된 조건식(예: 골든크로스, 거래량급증 등) 목록을 버튼 형태로 띄워 원하는 조건식만 실시간 매매 알고리즘에 온/오프 연동할 수 있습니다.
4. **조건 탐지/해제 이력 로그**: 실시간으로 조건 검색에 포착되거나 해제된 내역을 타임라인 로그로 적재합니다.
5. **체결 및 거래 이력 관리**: 자동 봇이나 사용자가 수동으로 매수/매도한 이력을 종목별 체결가, 체결수량, 매매구분, 체결 사유와 함께 직관적으로 정리합니다.
6. **보유 종목 관련 정보 영역**: 매수 완료된 주식 포트폴리오를 카드 뷰로 나열합니다. 각 카드에는 매입단가, 현재가, 평가금액, **최고 피크(PEAK) 수익률** 및 현재 수익률을 실시간 변동 폭에 따라 직관적으로 보여주며 즉시 매도 버튼이 포함되어 있습니다.
7. **거래 파라미터 설정**:
   * 조건 포착 시 자동 매수 기능 (On/Off)
   * 종목당 매수 예산 지정
   * 당일 최대 매수 한도
   * 거래 시간 제한 (매수 가능 시간 및 청산 마감 시간 설정)
   * 익절 트레일링 스톱 비율 (최고 PEAK점 대비 X% 하락 시 즉시 익절)
   * 손절선 비율 (매입가 대비 Y% 하락 시 즉시 손절)

## 🤖 샌드박스 시뮬레이터 (Simulator) 탑재
백엔드에는 장외 시간이나 API 키 연결이 되지 않은 상태에서도 자동 매매를 완벽하게 모의 테스트할 수 있는 **백그라운드 가동 엔진**이 내장되어 있습니다.
* 모의 시뮬레이션 활성화 시 가상의 주식 10종목의 가격이 매 2초마다 현실적으로 요동칩니다.
* 연동된 조건식에 따라 무작위로 종목 탐지 이벤트를 발생시키며, `자동 매수` 설정이 켜져 있으면 예산 범위 안에서 모의 매수를 수행합니다.
* 매수된 모의 잔고는 설정된 **손절 비율** 및 **트레일링 스톱 비율(최고 수익률 대비 하락 폭)** 계산에 따라 자동으로 매도 처리가 실행되어 대시보드로 실시간 피드백됩니다.

## 🚀 실행 및 테스트 방법

### 1. 전제 조건
* Node.js (V18 이상 권장) 및 Python 3.13 설치

### 2. 원클릭 통합 실행
가상환경 진입 및 프론트/백 서버 기동을 일괄 처리하기 위해 루트 폴더에서 PowerShell 스크립트를 실행합니다.

```powershell
.\run_app.ps1
```

* **백엔드(FastAPI)**: `http://127.0.0.1:8000`에서 실행됩니다. (PowerShell 프로세스로 별도 팝업)
* **프론트엔드(Vite + React)**: `http://localhost:5173`로 기동하며 웹 브라우저가 자동 연결됩니다.

## 🔧 개별 수동 실행 방법 (오류 발생 시)

PowerShell 프로세스 실행 제한 등의 이유로 스크립트 실행이 되지 않는 경우 두 개의 터미널을 열어 각각 실행해 주십시오.

#### 터미널 A (백엔드 FastAPI 실행)
```powershell
# 가상환경 활성화
.\venv\Scripts\Activate.ps1
# FastAPI 서버 실행
uvicorn src.app:app --port 8000 --reload
```

#### 터미널 B (프론트엔드 React 실행)
```powershell
cd frontend
npm run dev
```
